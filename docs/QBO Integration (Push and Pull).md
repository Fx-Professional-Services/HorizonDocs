Data push and pull functions in **Party > Customers**; **Orders > Items**; and **Accounting > Payment Terms**; **Tax Code**; **Invoices**. 

#### PUSH DATA (Horizon Party>Customers to QuickBooks) 


1. Before proceeding with the data push, check in QuickBooks if the data already exists. 

	- Type the name of the customer in the search bar. 
	- If there are no search results, proceed with the data push by following the steps below.

	![](https://github.com/Fx-Professional-Services/HorizonDocs/blob/main/assets/push_data_check_QuickBooks.png)

2. [Open FileMaker Pro Application](Open%20FileMaker%20Pro%20Application.md)
3.  [Log in to Horizon](Log%20in%20to%20Horizon.md)
4. On the main menu, click **Customers.**
![](https://github.com/Fx-Professional-Services/HorizonDocs/blob/main/assets/push_main_menu>customers.png)

5. Click on the name of the customer in the left pane. (You may need to scroll down.)
![](https://github.com/Fx-Professional-Services/HorizonDocs/blob/main/assets/push_customer's_name.png)

6. Click on **Push.**
![](https://github.com/Fx-Professional-Services/HorizonDocs/blob/main/assets/push_push_button.png)

7.  In QuickBooks, countercheck if the data has been pushed.

- Type the name of the customer in the search bar.

![](https://github.com/Fx-Professional-Services/HorizonDocs/blob/main/assets/push_customer's_name_search_bar.png)

- Click on the name of the customer in the search result.

![](https://github.com/Fx-Professional-Services/HorizonDocs/blob/main/assets/push_customer's_name_search_result.png)

- Customer’s record shows that data push has been successful.
![](https://github.com/Fx-Professional-Services/HorizonDocs/blob/main/assets/push_data_push_successful.png)

#### PULL DATA [QuickBooks to Horizon Accounting> Payment Terms]

1. [Open FileMaker Pro Application](Open%20FileMaker%20Pro%20Application.md)
2.  [Log in to Horizon](Log%20in%20to%20Horizon.md)
3. On the main menu, click on **Invoices.**

pull_main_menu>customers.png)

4. Click on **Payment Terms.**

![](https://github.com/Fx-Professional-Services/HorizonDocs/blob/main/assets/pull_invoices>payment%20terms.png)

Note: In the left pane, Net 15 Payment Term does not exist in the data base. To pull the data from QuickBooks, continue with the steps below.

5. Click on **Pull,** and wait until the data has been pulled. 

![](https://github.com/Fx-Professional-Services/HorizonDocs/blob/main/assets/pull_payment%20terms>pull%20button.png)

6. Data pull is successful if new data entries appear under **Payment Terms**.

![](https://github.com/Fx-Professional-Services/HorizonDocs/blob/main/assets/pull_succesful_data_pull.png)

Note: In the left pane, Net 15 Payment Term now exists in the data base.
